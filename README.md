# Placement Intelligence Platform

An AI-powered platform that helps engineering students prepare for campus placements.

## Features
- Resume upload and ATS scoring
- Skill gap analyser vs job descriptions
- Placement history analytics
- AI personalised roadmap generator
- AI mock interview module

## Tech Stack
- Frontend: Next.js 14, TypeScript, Tailwind CSS
- Backend: FastAPI (Python)
- Database: Supabase (PostgreSQL)
- AI: OpenAI GPT-4o
- Auth: Clerk
- Deployment: Vercel + Render
